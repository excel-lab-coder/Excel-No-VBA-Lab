⌨️ 全部覚える必要なし！実務で"本当に使う"ショートカットキー厳選集

💡 何十個も覚えるのは時間の無駄！

ネットで「Excel ショートカット 100選」といった記事をよく見かけますが、そのうち9割は実務でほとんど使いません（マウスでクリックしたほうが早い・脳のリソースの無駄遣いになるためです）。

ここでは、「これを覚えるだけでマウスを持ち替える回数が激減し、作業スピードが数倍になる」という、データ整形や集計で「息をするように使う」神ショートカットだけに厳選しました。

※超基本の Ctrl + C（コピー）、Ctrl + V（貼り付け）、Ctrl + Z（戻る）、Ctrl + S（上書き保存）は既に知っている前提で進めます。

【1】データの端まで一瞬で飛ぶ・選択する

何千行もあるデータベース表を扱う際、マウスのスクロールホイールを「シャーッ」と回している時間は最も無駄な時間です。これらは絶対にマスターしてください。

ショートカット 効果と使う場面
Ctrl + 矢印キー データの端っこまで一瞬でワープ（移動）する
空欄にぶつかる直前（最終行や右端）まで0.1秒でカーソルが飛びます。何万行でも一瞬です。
Ctrl + Shift + 矢印 データの端までを一気に[選択]する
先頭セルから最終行までをビーッとドラッグする地獄から解放されます。数式を下まで反映させたい時や、表全体をコピーしたい時の必須ワザです。
Ctrl + Home A1セル（一番左上）に一瞬で戻る
下の方で作業した後、「よし、次！」と表の頭に戻りたい時に使います。

【2】入力と数式編集のスピードを劇的に上げる

ここを押さえると、「Excelわかってる感」がグッと出ます。

ショートカット 効果と使う場面
F2 セル内の編集モードに入る
いちいちセルをダブルクリックする必要がなくなります。また、数式をチェックしている際にカーソルキーで動いてしまわないよう、セルの中に無理やり入り込むのにも最適です。
F4 直前の動作を[繰り返す] ＆ [絶対参照（$）の付与]
セル編集中、A1という文字の上で押すと `$A$1` → `A$1` と切り替わります（関数の必須操作）。
また、通常時は「さっき背景を黄色にしたから、このセル達にも同じ黄色を塗るぞ！」という繰り返し作業に使えます。
Ctrl + D 真上のセルを[真下]にそのままコピーする
Down（下）のD。上の数式や文字をそのまま下に持ってきたい時、コピペ（Ctrl+C→Ctrl+V）の2ステップを1ステップに短縮する神キーです。
Ctrl + R 左のセルを[右]にそのままコピーする
Right（右）のR。先ほどのCtrl+Dの右バージョンです。

【3】メニューを探す手間をなくす一発呼び出し

上部のリボンタブをあちこち探す時間をゼロにします。

ショートカット 効果と使う場面
Ctrl + 1 「セルの書式設定」ダイアログを一発で開く
右クリック → 下の方にある「セルの書式設定」を選ぶ…という手間を省略。桁区切りカンマを入れたり、日付の見た目を変えたりする際に1日100回は使います。
Ctrl + ; [今日の日付]を一瞬で入力する
台帳に「今日の作業日」を打つとき、「2026/03/18」などと手入力せずポンッと押すだけ。「れ」と書かれたキーです。

【4】Excel以外の作業も爆速化！Windows標準の神ショートカット

エクセルとブラウザ、フォルダを行き来する時間をゼロにする、全社会人必修のショートカットです。

ショートカット 効果と使う場面
Win + D [デスクトップ]を一瞬で表示する（全ウィンドウ最小化）
何十枚も開いたExcelやブラウザを一瞬でどかし、デスクトップにあるファイルを開きたい時に使います。もう一度押すと元の画面に戻ります。
Win + V [クリップボード履歴]から過去のコピーを貼る
「さっきコピーしたURLともう一つ前のテキストを…」と何度もウィンドウを行き来する必要がなくなります。直近のコピー履歴が一覧で出てくる超絶便利機能です。
Win + Shift + S [画面の一部]を瞬時に切り取る（スクショ）
エクセルのエラー画面や資料の一部だけを切り取って、そのままチャットツールやメールに「Ctrl+V」で直接貼り付けることができます。
Win + 矢印キー ウィンドウを[画面の右半分/左半分]にピタッと寄せる
左にブラウザ、右にエクセル…と参考資料を見ながら作業する際、ウィンドウの端をマウスでドラッグして調整する無駄な時間が完全に消えます。
✅ これだけで十分です！
これ以外のショートカット（例えばCtrl+Bで太字にする等）は、急いで覚える必要はありません。リボン上のアイコンをクリックした方が早い場面も多いからです。
まずはこのページで紹介した「移動系（Ctrl+矢印）」と「書式設定（Ctrl+1）」、「絶対参照（F4）」の3つを指に覚えさせるだけで、Excel操作のストレスは9割減ります。

当サイトは、VBA・マクロが使えない職場環境でも、Excel関数、Power Query、QRリーダー連携などを活用して、日々の手作業を減らすための実務向け情報を発信する個人運営サイトです。
